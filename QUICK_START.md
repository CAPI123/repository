# 🚀 Quick Start Guide

## Get Your Portfolio Running in 3 Steps

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Copy Images
```bash
npm run setup-images
```

### Step 3: Start Development Server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser!
